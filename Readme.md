## Instructions

To get started with this project, run

```bash
  git clone https://github.com/HrushiBorhade/DevLink.git
```
Install all the packages

```
npm install
```
create .env file with these variables:

```bash
DATABASE_URL=
NEXTAUTH_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

UPLOADTHING_SECRET=
UPLOADTHING_APP_ID=

REDIS_URL=
REDIS_SECRET=
```



Run the app
```
 `npm run dev`
```
